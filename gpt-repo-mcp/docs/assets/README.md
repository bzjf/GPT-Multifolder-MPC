# Documentation Assets

Place GitHub README and setup screenshots in this directory.

Current connector setup images:

- `chatgpt-server-url.png`: ChatGPT Developer Mode "New App" form with **Server URL** selected.
- `chatgpt-tunnel-id.png`: ChatGPT Developer Mode "New App" form with **Tunnel** selected.
- `chatgpt-server-url.svg`: ChatGPT Developer Mode "New App" form with **Server URL** selected.
- `chatgpt-tunnel-id.svg`: ChatGPT Developer Mode "New App" form with **Tunnel** selected.

The PNGs are used by the docs. The SVGs are sanitized source mockups. If replacing them with real screenshots, keep screenshots small, cropped to the relevant UI, and free of real tunnel URLs, tunnel ids, tokens, repo paths, account names, or other private data.
